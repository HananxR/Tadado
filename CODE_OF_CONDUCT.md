# 行为准则

## 我们的承诺

为了营造开放和包容的社区环境，我们承诺尊重所有参与者，无论其经验水平、性别、性取向、残疾状况、外貌、种族或宗教信仰。

## 我们的标准

### 鼓励的行为

- 友好、包容的语言
- 尊重不同的观点和经验
- 建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表达同理心

### 不可接受的行为

- 挑衅、侮辱或贬损性言论
- 骚扰行为（公开或私下）
- 未经许可发布他人隐私信息
- 其他在专业环境中被合理视为不适当的行为

## 执行

社区维护者有责任澄清和执行行为准则，并对不当行为采取适当且公平的纠正措施。
