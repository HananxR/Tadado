# 截图目录

请将以下 5 张软件截图放入此目录，文件名严格匹配：

| 文件名 | 内容 |
|--------|------|
| `task-view.png` | 任务视图全貌 — 9列表格 + 编辑面板 + 优先级整行渲染 + 状态栏 |
| `task-input.png` | Markdown 任务输入框 — 展示完整语法输入 |
| `heatmap.png` | 日历热力图 — 12月矩阵 + 标签云 + 活动报告 |
| `batch-view.png` | 批量管理控制台 — 全宽表格 + 批量工具栏 + 标签管理面板 |
| `demo-space.png` | 演示空间 — 分区切换下拉 + 15个演示任务一览 |

截图后 Push 到 GitHub，README 中的 `raw.githubusercontent.com` 链接即自动生效。

**建议截图方式**：
1. 启动 Tadado，切换到「演示空间」分区
2. `Win+Shift+S` 截图，保存为 PNG
3. 图片宽度建议 ≥ 1440px，确保在高清屏上清晰
