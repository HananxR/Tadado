1. 快速入门

1.1 安装与启动

Tadado 为 Windows 桌面应用，提供安装包（Setup.exe）一键安装。安装完成后双击桌面快捷方式即可启动，程序会在系统托盘显示图标，主窗口同时打开。

数据存储在本地 SQLite 数据库文件中，无需联网，无需注册账号。

1.2 创建第一个任务

在编辑视图（默认首页）顶部的输入框中，按以下 Markdown 语法输入任务：

- [ ] TODO 2026-06-15 完成使用手册编写 #文档 #技术

格式说明：

组成部分说明示例
- [ ]任务标记（固定前缀）- [ ]
状态TODO / DOING / DONE / OVERDUETODO
截止日期YYYY-MM-DD 格式，可选2026-06-15
标题任务名称，必填完成使用手册编写
标签# 开头，空格分隔，可选#文档

按 Enter 键创建任务。输入框带有红色闪烁提示——当语法格式不正确时会闪烁提醒。

💡 提示输入框右侧有"新建多任务"按钮，可快速创建多行模板任务；创建日期会自动填充为当天。

1.3 界面概览

Tadado 有三个主要视图，通过标题栏按钮切换：

视图用途
编辑视图日常任务增删改查，快速编辑
活动分析日历热力图、活动统计、标签浏览
任务管理高级筛选、批量操作、标签管理

标题栏最左侧为 App 图标按钮，点击可返回编辑视图首页，同时清空筛选条件并刷新数据，是各视图间快速归位的入口。

窗口为无边框设计，拖动顶部标题栏空白区域可移动窗口，边缘 6px 区域可拖拽缩放。

2. 任务编辑视图

2.1 新建任务

新建单任务：在顶部输入框输入 Markdown 语法后按 Enter，或点击标题栏 新建单任务 按钮。草稿模式下输入框预填当天日期和标签模板，蓝色高亮提示"未保存的草稿"。

新建多任务：点击标题栏 新建多任务 按钮，编辑器区会生成 3 行不同模板的任务，统一加粗样式。所有多任务创建后统一标记为粗体。

2.2 任务列表

任务以表格形式展示，包含以下列：

  • 复选框 — 点击可切换完成状态（TODO ↔ DONE）
  • 序号 — 行号
  • 创建时间 — 任务创建日期
  • 任务内容 — 标题和标签摘要（自适应宽度）
  • 截止日期 — 到期时间
  • 进度 — 完成百分比（0-100%）
  • 状态 — 彩色状态徽章（OVERDUE=红 / DOING=橙 / TODO=蓝 / DONE=绿）
  • 标签 — 标签列表

任务行支持交替行颜色，悬停高亮。挂起的任务以 45% 透明度显示。当前活动任务以红色粗体高亮。

列表底部有分页控件，可切换每页 20/50/100 条，翻页浏览。

2.3 编辑面板

点击列表中的任务，右侧弹出编辑面板：

  • Markdown 源码编辑 — 直接编辑 raw_md，右侧实时 HTML 预览
  • 截止日期 — 日历选择器 + 时间选择器，可快速计算相对日期
  • 活动时间线 — 状态变更 + 进度记录的时间轴，可追加新进度
  • 分区选择 — 下拉切换任务所属分区

编辑后点击 保存 提交更改，删除 移除任务。

2.4 筛选与排序

列表上方筛选栏提供：

  • 全文搜索 — 输入关键词实时搜索（基于 SQLite FTS5，300ms 防抖）
  • 状态过滤 — 下拉选择 TODO / DOING / DONE / OVERDUE
  • 排序方式 — 按紧急度 / 截止日期 / 创建时间 / 状态 / 标题排序

2.5 统计栏

  • 状态徽章条 — 显示各状态任务计数（OVERDUE | TODO | DOING | DONE），点击可快速筛选
  • 进度动态条 — 6 个时间段按钮（昨天/今天/上周/本周/上月/本月），展示活动最频繁的任务轮播
  • 速览栏 — 按紧急度排序的任务卡片自动轮播（5 秒切换），点击可跳转编辑

3. 活动分析视图

3.1 日历热力图

12 个月 × 7 天 × 5 周的紧凑型日历热力图，用颜色深浅表示每日活动量：

  • 鼠标悬停单元格可查看：日期、活动记录数、涉及任务数、标签分布
  • 顶部 ◀ 年份 ▶ 按钮切换年份
  • 下拉菜单可筛选特定标签对应的活动

3.2 统计面板

热力图下方展示汇总统计：总记录数 / 活跃天数 / 最长连续天数 / 当前连续天数 / 日均 / 月均 / 周均。

3.3 任务树与内容浏览

左侧为标签云面板，勾选标签后右侧显示对应任务的活动内容：

  • 搜索框实时过滤标签
  • 导航栏可逐条浏览活动记录
  • 支持将活动报告导出为 Markdown / Excel / TXT 格式

4. 任务管理控制台

4.1 高级筛选

左侧筛选面板（180px）提供比编辑视图更精细的过滤条件：

  • 关键词搜索
  • 状态过滤
  • 创建日期范围 — 起止日期可通过日历选择
  • 截止日期范围 — 同上
  • 进度范围 — 0% / 1-25% / 26-50% / 51-75% / 100%
  • 标签过滤 — 空格分隔多个标签
  • 归档状态 — 全部 / 未归档 / 已归档

4.2 批量操作

勾选任务后，工具栏提供：

操作说明
批量状态变更将选中任务设为 DOING 或 DONE
批量删除删除选中任务（需确认）
批量挂起挂起选中任务（半透明显示）
批量重启恢复已挂起的任务
延后处理批量推迟截止日期 +1/+2/+5/+7/+10 天
导出导出选中任务为 Markdown 或 Excel
全选 / 取消全选一键操作所有可见任务
⚠ 注意批量删除和延后处理会直接影响截止日期和任务数据，操作前请确认选中范围。

4.3 标签管理面板

右侧面板显示当前分区所有标签及其使用次数：

  • 重命名标签 — 选中一个标签 → 输入新名称 → 所有使用该标签的任务自动更新
  • 合并标签 — 多选标签 → 选择合并目标 → 源标签替换为目标标签后删除（不可撤销）
  • 搜索标签 — 实时过滤标签列表

4.4 归档管理

  • 手动归档 — 点击 归档已完成 立即归档当前分区所有 DONE 状态任务
  • 清除已归档 — 点击 清除已归档 永久删除所有已归档任务（需二次确认）
⚠ 注意手动归档会忽略设置的归档天数阈值，直接归档分区内所有已完成任务。清除已归档不可恢复。

5. 分区管理

5.1 什么是分区

分区是独立的任务空间，不同分区的任务、标签、设置互不影响。默认提供三个分区：工作、个人、学习。

5.2 切换分区

点击状态栏最左侧的分区名称按钮，从下拉菜单选择目标分区。当前分区以粗体高亮。

5.3 密码保护

可在设置中为每个分区单独设置密码（SHA-256 哈希存储）：

  • 锁定分区后，切换到该分区时显示锁定遮罩，需输入密码解锁
  • 解锁状态仅在内存中保持，关闭程序后重新锁定
  • 自动锁定 — 在设置中配置空闲超时（1/5/10/30/60 分钟），超时后自动锁定当前分区

5.4 管理分区

在设置对话框的"归档"标签页中，可：新建分区 / 重命名分区 / 删除分区（分区内有任务时不可删除）/ 设为默认分区。

6. 系统托盘与提醒

6.1 系统托盘

程序启动后在 Windows 系统托盘显示图标：

  • 右键菜单 — 显示/隐藏窗口、新建单任务、新建多任务、退出
  • 双击图标 — 切换窗口显示/隐藏
  • 最小化行为 — 根据设置，点最小化按钮可隐藏到托盘或保留在任务栏

标题栏提供四个窗口控制按钮：隐藏到托盘 / 最小化 / 切换全屏 / 关闭。

6.2 任务提醒

后台定时检查任务状态（每分钟一次）：

  • 检测到期和即将到期的任务
  • 通过 Windows 系统通知气泡提醒
  • 可在设置中开启/关闭提醒、配置检查间隔

6.3 免打扰时段

在设置中配置安静时间范围（如 22:00-08:00），该时段内不发送通知。

6.4 循环任务

当循环任务（在 raw_md 中设置循环规则，如 +1d / +1w / +1m / +1y）标记为 DONE 后，系统自动创建下一次实例（TODO 状态、截止日期按规则偏移）。

7. 导出

7.1 Markdown 导出

在编辑视图或任务管理控制台中，选中需要导出的任务，点击批量工具栏的导出下拉菜单，选择 导出 Markdown。每个任务输出一行 Markdown 格式文本，保存为 .md 文件。

7.2 Excel 导出

同样在导出下拉菜单中选择 导出 Excel，生成多列表格文件（序号 / 任务 / 状态 / 进度 / 信息等），可用 Excel 或 WPS 打开。

7.3 活动报告导出

在活动分析视图右侧内容浏览区，点击导出按钮可将当前标签对应的活动记录导出为 Markdown / Excel / TXT 三种格式之一。文件名自动生成，包含分区名、日期范围和标签数量。

8. 设置

点击标题栏 设置 按钮打开设置对话框，包含以下配置项：

8.1 外观

  • 主题 — 亮色 / 暗色 / 跟随系统
  • 最小化到托盘 — 开启后点击最小化按钮隐藏到托盘而非任务栏
  • 自动锁定超时 — 1/5/10/30/60 分钟，超时后自动锁定当前分区

8.2 任务列表

  • 每页条数 — 20 / 50 / 100
  • 默认排序 — 按紧急度 / 截止日期 / 创建时间 / 状态 / 标题
  • 热力图起始年份 — 日历热力图的数据起始年份

8.3 提醒

  • 启用提醒 — 开启/关闭任务到期提醒
  • 检查间隔 — 提醒检查频率（小时）
  • 免打扰开始 / 结束 — HH:MM 格式，该时段内不弹通知

8.4 归档

  • 分区列表 — 显示所有分区的名称、默认标记、可见性、自动归档开关、归档天数
  • 右键菜单 — 设为默认 / 重命名 / 删除分区
  • 归档天数 — 任务完成后多少天自动归档（0=立即，9999=永不）

8.5 MOTD

4 条自定义励志语录，在状态栏循环显示。

9. 常见问题与故障排除

Q: 窗口显示异常（空白、错位、无法拖动）

尝试以下步骤：

  1. 关闭并重新启动应用
  2. 如果窗口超出屏幕，可尝试按 Win+← 或 Win+→ 将窗口吸附到屏幕一侧
  3. 切换全屏模式再退出（标题栏右三按钮）
  4. 如果问题持续，删除 resources/config.json 重置配置（注意备份）

Q: 搜索不到任务

  1. 确认当前所在分区是否正确（状态栏最左侧分区名）
  2. 检查筛选条件是否过于严格（状态过滤、日期范围等）
  3. 确认任务未被归档（已归档任务在编辑视图不显示，需到任务管理控制台查看）
  4. 检查搜索关键词是否匹配任务标题或标签

Q: 提醒不生效

  1. 确认设置中"启用提醒"已勾选
  2. 检查当前时间是否在免打扰时段内
  3. Windows 系统设置 → 通知 → 确认 Tadado 的通知权限已开启
  4. 确认任务确实设置了截止日期且已到期或即将到期

Q: Markdown 语法解析失败

输入框闪烁红色表示语法解析失败。请检查：

  1. 是否以 - [ ] 开头（注意空格）
  2. 状态词是否正确：TODO / DOING / DONE
  3. 日期格式是否为 YYYY-MM-DD（如 2026-06-15）
  4. 标签是否以 # 开头、空格分隔

Q: 数据库文件在哪里？如何备份？

数据库文件位于程序安装目录下的 resources/tadado.data（SQLite 格式）。备份方式：

  1. 关闭 Tadado
  2. 复制 resources/tadado.data 到安全位置
  3. 如需恢复，将备份文件覆盖同名文件即可
💡 建议定期备份数据库文件，特别是在批量操作或标签合并前。

Q: 数据丢失或数据库损坏

  1. 检查 resources/ 目录下是否有 tadado.data.bak 备份文件
  2. 如有备份，将 .bak 重命名为 tadado.data 替换损坏文件
  3. 如无备份，建议在后续使用中定期手动备份

Q: 程序无法启动

  1. 确认系统为 Windows 10 或更高版本
  2. 尝试以管理员权限运行
  3. 检查杀毒软件是否拦截了程序
  4. 重新安装最新版本（覆盖安装不会丢失数据）
