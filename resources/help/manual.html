1. 快速入门

1.1 安装与启动

Tadado 提供 Windows 安装包一键安装。启动后程序常驻系统托盘，主窗口同步打开。所有数据存储在本地 SQLite 文件，无需联网、无需注册。

1.2 创建任务

在任务视图顶部的输入框中输入 Markdown 格式的任务：

- [   ] 2026-06-15 完成使用手册编写 #文档

格式说明：

部分说明示例
- [ ]任务前缀 + 优先级（* 数量 0-3，越紧急越多）- [***]（紧急）
截止日期YYYY-MM-DD，可选2026-06-15
任务标题必填，描述任务内容完成使用手册编写
标签# 开头，空格分隔，可选#文档
提示优先级通过 - [ ] 中 * 的数量表示：[ ]=普通、[* ]=关注、[** ]=重要、[***]=紧急。

1.3 界面概览

Tadado 有三个视图，通过标题栏按钮切换：

视图用途
任务视图日常任务的增删改查
活动分析日历热力图、活动统计
任务管理批量操作、标签管理、归档

窗口为无边框设计，拖动标题栏空白区域可移动，边缘 6px 区域可拖拽缩放。标题栏左侧 App 图标点击可返回任务视图并清空筛选。

2. 任务视图

2.1 新建任务

  • 新建单任务 — 输入框输入 Markdown 后按 Enter，或点击标题栏 新建单任务
  • 新建多任务 — 点击标题栏 新建多任务，编辑器生成 3 行模板任务
  • 草稿模式 — 新建时输入框蓝色高亮，未保存的草稿可丢弃

2.2 任务列表

9 列表格展示：复选框 / 序号 / 创建时间 / 任务内容 / 截止时间 / 进度 / 状态 / 标签 / 归档。

  • 优先级渲染 — 整行背景色标识：红（紧急）→ 橙（重要）→ 绿（关注）→ 淡蓝（普通）
  • 状态徽章 — 待办 / 进行中 / 已完成 / 已逾期，彩色区分
  • 行选择 — 单选/多选，选中任务红色加粗凸显
  • 分页 — 每页 20/50/100 条
  • 全选 — 工具栏全选按钮，配合批量操作

2.3 编辑任务

点击任务行，右侧弹出编辑面板：

  • Markdown 源码编辑 — 直接编辑原始 Markdown，右侧实时预览
  • 截止日期 — 日历选择器 + 时间选择器
  • 活动时间线 — 任务进展记录，支持追加新进度
  • 优先级 — 下拉切换紧急程度
  • 分区选择 — 将任务迁移到其他分区

编辑后点击 保存 提交，删除 移除任务。

2.4 筛选与排序

  • 全文搜索 — SQLite FTS5，实时匹配
  • 状态过滤 — 待办 / 进行中 / 已完成 / 已逾期
  • 优先级过滤 — 紧急 / 重要 / 关注 / 普通
  • 排序 — 优先级 → 截止日期 → 创建时间

2.5 右键菜单

在任务行上右键可快速：更改状态 / 更改优先级 / 调整分区 / 延后处理 / 中止 / 重启 / 删除。支持多选批量操作。

2.6 统计与速览

  • 状态徽章条 — 各状态任务计数，点击快速筛选
  • 进度动态条 — 6 个时间段按钮（昨天/今天/上周/本周/上月/本月），显示最近活跃任务的最新进展
  • 速览栏 — 6 个预设按钮（昨天/今天/上周/本周/上月/本月）+ 自动轮播任务卡片（每 5 秒切换，每组 3 个）

速览栏过滤逻辑

点击预设按钮后，任务列表按 创建时间 ≤ 该日上限 过滤，并排除已完成（DONE）的任务：

按钮含义
今天今天及之前创建的 + 还没做完的
昨天昨天及之前创建的 + 还没做完的
本周本周日及之前创建的 + 还没做完的
上周上周日及之前创建的 + 还没做完的
本月本月最后一天及之前创建的 + 还没做完的
上月上月最后一天及之前创建的 + 还没做完的

不再按任务截止日期过滤，确保所有未完成的任务都能在对应时间窗口内看到。

3. 活动分析

3.1 日历热力图

12 个月 × 7 天 × 5 周的紧凑日历，颜色深浅表示每日活动量：

  • 悬浮单元格查看：日期、活动数、涉及任务、标签分布
  • 顶部 ◀ 年份 ▶ 切换年份
  • 下拉筛选特定标签的活动

3.2 标签浏览

标签云面板：勾选标签后右侧显示对应活动内容，支持搜索过滤和导航栏逐条浏览。

3.3 活动报告

导出当前标签的活动记录为 Markdown / Excel / TXT。

4. 任务管理控制台

4.1 筛选面板

左侧面板提供：关键词搜索 / 状态过滤 / 优先级过滤。

4.2 批量操作

工具栏提供全选 + 导出（Markdown / Excel）+ 已选计数。通过右键菜单可对多选任务执行：

  • 更改状态 / 更改优先级 / 调整分区 / 延后处理 / 中止 / 重启 / 删除
注意批量删除和延后处理直接影响任务数据，操作前请确认选中范围。调整分区需验证目标分区密码。

4.3 标签管理

右侧面板展示当前分区所有标签及其使用次数：

  • 重命名 — 选中标签 → 输入新名称 → 全局自动更新
  • 合并 — 多选标签 → 选择目标 → 源标签替换为目标后删除
  • 搜索 — 实时过滤标签列表
注意标签合并不可撤销，操作前务必确认。

4.4 归档管理

  • 手动归档 — 点击 归档已完成 立即归档当前分区所有 DONE 任务
  • 清除已归档 — 点击 清除已归档 永久删除归档任务（需二次确认）

5. 分区管理

5.1 什么是分区

分区是独立的任务空间，不同分区的任务、标签互不影响。默认提供：工作、学习、个人。

5.2 切换分区

点击状态栏最左侧的分区名称 → 下拉菜单选择目标分区。当前分区以 ✓ 标记。

5.3 密码保护

  • 在设置中为每个分区单独设置密码
  • 锁定分区后，切换到该分区需输入密码
  • 解锁状态仅内存保持，关闭程序后重置
  • 自动锁定 — 按分区独立设定空闲超时（1/5/10/30/60 分钟）

5.4 管理分区

在设置对话框的分区表格中：新建 / 重命名 / 设密码 / 删除（分区内有任务时不可删除）。

6. 系统托盘与提醒

6.1 托盘操作

  • 右键菜单 — 显示/隐藏窗口、新建单任务、新建多任务、退出
  • 双击图标 — 切换窗口显示/隐藏
  • 最小化 — 开启"最小化到托盘"后，最小化隐藏到托盘而非任务栏

6.2 任务提醒

  • 后台每分钟检查到期任务
  • Windows 系统通知气泡提醒
  • 可在设置中开启/关闭、配置检查间隔

6.3 免打扰

设置安静时段（如 22:00—08:00），该时段内不弹出通知。

7. 设置

7.1 外观

  • 主题 — 亮色 / 暗色
  • 最小化到托盘 — 开启后最小化按钮隐藏到托盘

7.2 任务列表

  • 每页条数 — 20 / 50 / 100
  • 默认排序 — 优先级 / 截止日期 / 创建时间 / 状态 / 标题
  • 热力图起始年份 — 日历热力图的数据起始年

7.3 提醒

  • 启用提醒 — 开启/关闭
  • 检查间隔 — 小时
  • 免打扰开始/结束 — HH:MM 格式

7.4 分区管理

  • 分区表格：名称 / 默认分区 / 可见 / 自动归档 / 归档阈值(天) / 自动锁定(分) / 密码
  • 右键菜单：设默认 / 重命名 / 设密码 / 删除

7.5 MOTD

自定义励志语录，状态栏循环显示。

8. 常见问题

Q: 窗口显示异常（空白、错位、无法拖动）

  1. 关闭并重启应用
  2. 切换全屏再退出（标题栏右侧按钮），窗口会重置到屏幕中央

Q: 搜索不到任务

  1. 确认当前所在分区（状态栏左侧分区名）
  2. 检查筛选条件是否过严
  3. 确认任务未被归档（归档任务在编辑视图不显示）

Q: 提醒不生效

  1. 确认设置中"启用提醒"已勾选
  2. 检查是否在免打扰时段内
  3. Windows 设置 → 通知 → 确认 Tadado 权限已开启

Q: 如何备份数据

  1. 关闭 Tadado
  2. 复制 resources/tadado.data 到安全位置
  3. 恢复时将备份文件覆盖同名文件

Q: 程序无法启动

  1. 确认系统为 Windows 10 或更高
  2. 检查杀毒软件是否拦截
  3. 重新安装最新版本（覆盖安装不丢失数据）
