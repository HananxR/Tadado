# Changelog

Tadado 版本更新日志。格式参考 [Keep a Changelog](https://keepachangelog.com/)。

---

## [0.2.0] — 2026-06-12

### Changed
- **状态下拉框即时刷新进度**：切换至"已完成"进度自动刷新为 100，切回"进行中"自动设为 80，无需点击"追加进展"
- **截止时间默认值对齐快速计算**：新建任务模板中的截止时间从当前时刻改为快速计算默认值 23:59
- **创建任务默认截止日**：新建任务预填当天日期（对齐 DESIGN.md），移除"最近周五"逻辑
- **快速计算标签文案**："任务类型" → "时间粒度"，"临时" → "天"

### Fixed
- 进度 100% 因列宽不足显示为"00"：表格进度列与输入框拓宽至 55px
- 多条状态→已完成路径遗漏进度=100 自动设定：双击编辑对话框、批量创建、快速输入栏、Markdown 保存均补齐
- 批量更新状态至已完成时同步设置 completed_at 和进度字段
- 单任务保存时截止时间缺 fallback，与多任务保存行为不一致

## [0.1.2.3] — 2026-06-11

### Changed
- **提醒重构**：取消托盘弹窗，提醒信息融入速览栏与进度栏轮播展示
- 速览栏标签时间粒度细化：今天到期的任务显示"X 分钟后/X 小时后/HH:MM 截止/已超时"
- 进度栏 hint/轮播模式融合截止紧迫度标识，逾期/今日任务排序加权
- 进度栏 hint 模式支持点击跳转任务，轮播数据与左侧列表同步

### Added
- **关于对话框升级日志**：版本号下方展示当前版本升级内容，层级分明
- 关于对话框新增下载渠道区块（GitHub Releases / 阿里云盘）
- 检查更新结果内联到版本行（检查中 ··· / ✓ 已是最新 / 🆕 新版本）
- 欢迎页 Banner 动态适配分区状态：空分区显示欢迎语，有任务显示日历日期

### Fixed
- Banner 背景文字可读性：新增半透明遮罩层，副标题颜色对比度提升
- 更新检查顺序调整为阿里云盘优先、GitHub 备用，适配国内网络
- 抑制 QSslSocket 设备未打开警告日志
- 启动遮罩期间强制箭头光标，消除鼠标转圈
